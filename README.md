Human sleep isn't a linear progression; it occurs in cycles. A typical cycle lasts roughly 90 minutes.

Waking up during the middle of a cycle (especially Stage 3 Deep Sleep) causes Sleep Inertiaâ€”that groggy, disoriented feeling that can last for hours. Somnus uses the formula:

$$T_{wake} = T_{sleep} + (n \times 90) + T_{latency}$$

Where $n$ is the number of cycles (ideally 5 or 6) and $T_{latency}$ is the time it takes you to actually drift off.
